
# README

Этот бот на Python помогает пользователю выбрать карьерное направление по трем категориям: творческий, технический и социальный.

## Описание

Бот использует библиотеку `telebot` для работы с Telegram API. После запуска при команде `/start` бот приветствует пользователя и предлагает написать одно из ключевых слов: "творческий", "технический" или "социальный". В ответ бот выводит список профессий из выбранной категории.

## Как пользоваться

1. Запустить бота.
2. Отправить команду `/start`.
3. Ввести одно из слов:  
   - творческий  
   - технический  
   - социальный  
4. Получить список профессий соответствующего карьерного направления.

## Структура кода

- Импортируется токен доступа из файла `config.py`.
- Словарь `careers` содержит категории и списки профессий.
- Обработчик команды `/start` отправляет приветствие.
- Обработчик любых сообщений проверяет введённое слово и выводит соответствующий список профессий или просит повторить ввод.
